# Personal Portfolio Website
A modern, responsive portfolio website built using React, HTML, Tailwind CSS, and JavaScript. Designed with smooth animations, clean UI, and a focus on performance and user experience.

### Features
- Smooth and modern UI/UX
- Fully responsive design (mobile, tablet, desktop)
- Fast and optimized performance
- Styled with Tailwind CSS
- Interactive navigation and sections
- Projects showcase with live previews
- Resume download functionality
- Clean and minimal design
